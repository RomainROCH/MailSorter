### Personal Use (Free)

For personal/non-commercial use, you can manually configure the native messaging host:

1. Create `backend/app_manifest.json` with the correct path to `main.py`
2. Register the manifest with your browser/email client

See the [User Guide](../docs/USER_GUIDE.md) for detailed manual setup instructions.
